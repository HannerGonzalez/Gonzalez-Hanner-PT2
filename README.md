## Descripción

El proyecto el cuál consiste en instalar y configurar Nextcloud para utilizarlo como un sistema de almacenamiento en la nube. A través de la plataforma podremos subir archivos, crear carpetas y organizarlas. También se crearán nuevos usuarios y se les asignarán roles o permisos según las tareas que deban realizar dentro del sistema. En conjunto, el objetivo principal del proyecto será ofrecer un entorno seguro y colaborativo para gestionar y compartir información.
